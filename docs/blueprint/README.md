# Blueprint Development

This project uses Blueprint for structured development documentation and AI-assisted workflows.

## Directory Structure

- `manifest.json` - Blueprint configuration and version tracking
- `feature-tracker.json` - Feature implementation progress tracking
- `work-orders/` - Task packages for subagent execution
- `ai_docs/` - Curated documentation for AI context

## Related Directories

- `../prds/` - Product Requirements Documents
- `../adrs/` - Architecture Decision Records
- `../prps/` - Product Requirement Prompts

## Commands

Run `/blueprint:status` to check configuration and available commands.
